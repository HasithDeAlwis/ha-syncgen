# Roadmap

The future development plans for ha-syncgen and what's coming next.

## Current Status: Alpha (v0.1.0)

ha-syncgen is currently in early development with core functionality implemented for PostgreSQL streaming replication automation.

<Callout type="info">
  The tool successfully generates working PostgreSQL HA scripts and configurations, but is not yet production-ready.
</Callout>
